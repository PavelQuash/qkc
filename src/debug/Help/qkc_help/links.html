Ссылки на подобные ресурсы

В данном разделе представлены ссылки на различные заинтересовавшие меня ресурсы хоть как-то относящиеся к qshKeyCommander, программы подобного qshKeyCommander типа и программы, которыми он может управлять. На момент публикации ссылки существовали, дальнейшая их "жизнь" не отслеживается.

qshKeyCommander - домашняя страничка программы, где можно скачать последнюю версию, узнать новости и расширить её функциональность с помощью плагинов.

DreamKeys - программа очень похожая на qshKeyCommander, но имеющая гораздо меньше функций, так как обновлялась сравнительно давно.

EscapeClose - программа, позволяющая по нажатию клавиши Esc закрывать активное окно или устанавливать курсор мыши в правый верхний угол активного окна на крестик; не сказал бы, что эта программа маленькая, но для такого рода утилиток вполне сойдёт, хотя можно сделать и в рамках 5 килобайт; программа при своих, практически никаких, возможностях предлагает зарегистрироваться; в чисто ознакомительных целях посмотреть можно...

Hot Keyboard - неплохая Shareware программа для управления системой, слишком много наворотов и всё это в одном флаконе... Тем более используется слишком много диалогов, к которым не сразу привыкаешь.

Light Alloy - очень удобная программа для просмотра видео файлов, она сама поддерживает WinLIRC, но в отслеживании кнопок есть небольшие недочёты, которых нет в управляющем плагине, а при переустановке LightAlloy приходится заново всё переустанавливать, в случае с qshKeyCommander'ом всё хранится в одном файле инициализации и нет никаких проблем ;).

LIRC - ссылка на оригинальный сайт LIRC для Linux, содержит множество конфигурационных файлов для ПДУ, описаний схем и другой документации.

UPX - программа предназначена для архивирования исполняемых файлов, их ресурсов, и загрузки в память. Отличный паковщик exe, dll.

Winamp - плеер всех времён и народов; qshKeyCommander с лёгкостью может им управлять; также для управления Winamp существует множество плагинов:
  • Album List - плагин, позволяющий искать mp3 по указанным каталогам и сортирующий песни по альбомам; встраивается в Winamp, создаёт дополнительное окно типа Playlist со списком найденных альбомов; как и сам Winamp позволяет управлять собой с помощью API сообщений; для управления этим плагином можно скачать одноимённый модуль для qshKeyCommander;
  • Hotamp - плагин для управления Winamp2 с помощью горячих клавиш, имеет Shareware статус :(, притом существует множество программ и плагинов, которые не берут за их работу ни копейки;
WinLIRC - это программа, предназначена для отслеживания сигналов от пульта дистанционного управления и других источников для Windows платформы, здесь можно найти описание её работы, ссылки на программы, которые с ней работают и другую полезную информацию.
